# 標註規範（Annotation Guideline）

| 版本 | v0.2 |
|------|------|
| 適用任務 | CanopySeg、LeafInst、LeafDefect |
| 工具 | LabelMe、CVAT、Roboflow（擇一，主推 CVAT） |

---

## 壹、總則

| 原則 | 說明 |
|------|------|
| 一致性 | 同一標準下不同人應標出相近結果（IoU ≥ 0.85 為合格） |
| 精準度 | 多邊形邊緣須沿物件實際輪廓，不可粗描 |
| 完整性 | 不遺漏可辨識的目標；不確定時加 `is_uncertain=true` 而非略過 |
| 可追溯 | 每筆標註記錄標註者、版本、時間 |

## 貳、樹冠影像標註（CanopySeg）

### 一、類別

| 類別 | 中文 | 定義 |
|------|------|------|
| `canopy_other` | 樹冠非植生 | 樹冠輪廓內的枝幹、空隙、陰影 |
| `veg_green` | 綠葉 | 健康綠色葉片 |
| `veg_yellow` | 黃化葉 | 明顯黃化（落葉前期、缺氮、病害） |
| `veg_brown` | 褐化葉 | 乾枯、褐變、枯死 |

> 背景（`background`）為其餘所有像素，不主動標註。

### 二、邊界規則

- **樹冠輪廓**：以肉眼可辨識的最外圈葉片連線為界，不含明顯獨立伸出的單枝
- **樹冠重疊**：若主目標樹冠被其他樹冠遮擋，僅標可見部分，加 `occluded=true`
- **天空**：留白為背景，不標
- **建物 / 電線**：留白為背景

### 三、判讀色彩標準

> 標註者須在 sRGB 螢幕、色溫 6500K、亮度 ≥ 250 cd/m²、無強反射的環境下標註。

| 類別 | 主觀判讀 | 客觀色相（HSV） |
|------|----------|------------------|
| veg_green | 飽和綠 | H ∈ [70, 160], S ≥ 30, V ≥ 30 |
| veg_yellow | 黃→黃綠 | H ∈ [40, 70] |
| veg_brown | 褐、紅褐 | H ∈ [10, 40], S ≥ 20 |

> 客觀數值只供參考；最終以人類判讀為準（避免色卡偏差時誤標）。

## 參、葉片影像標註（LeafInst）

### 一、目標

每片**完整可見**的葉片獨立標註為一個 instance。

### 二、規則

- **完整可見**：葉片邊緣至少 90% 連續可見；缺角不影響
- **重疊**：兩葉重疊時各自標各自，不共享像素（按 z 序最上層歸誰）
- **葉柄**（依 [ADR-0001 議題 2](decisions/0001-open-questions.md#議題-2葉片標註單位全葉-vs-全葉--葉柄)）：**葉柄併入葉片實例**，邊界沿葉柄外緣描繪；不另設 `petiole_json` 欄位、`leaf_instance` 表保留單一 `polygon_json`
- **病葉**：仍標為 leaf；其上的病徵在 LeafDefect 階段標
- **複葉**：以**小葉**為單位；同一複葉的所有小葉指向同一 `compound_leaf_id`

### 三、尺度物標註

| 類型 | 標註方式 | ref_size_mm |
|------|----------|-------------|
| `a4_grid` | 標 A4 方格紙的最外框 bbox | 297（長邊） |
| `coin_5twd` | 標 5 元硬幣外圓 bbox | 22.0 |
| `coin_10twd` | 標 10 元硬幣外圓 bbox | 26.0 |
| `ruler` | 標尺所示 1 cm 區間 | 10.0 |

> 每張葉片影像至少標一個尺度物，否則影像不可進入 LeafInst 訓練集。

## 肆、葉片病徵標註（LeafDefect）

### 一、類別（多標籤，可同一像素疊加）

| 類別 | 中文 | 定義 |
|------|------|------|
| `lesion` | 病斑 | 形狀界限明顯的病斑（褐斑、黑點、白粉斑） |
| `chlorosis` | 黃化 | 漸層黃化、缺綠（不同於 lesion 邊界明確） |
| `necrosis` | 壞死 | 組織枯死、深褐黑色塊 |
| `hole` | 破洞 / 缺損 | 葉片實際缺失的部分 |

### 二、邊界規則

- **lesion**：沿病斑外緣描，不擴邊
- **chlorosis**：以肉眼可辨的色變為界；漸層處取「中間色」線
- **necrosis**：取最深色區域為主，邊緣與 lesion 重疊時兩者皆標
- **hole**：標葉片缺失的多邊形（葉脈外露區也算）

### 三、不標的東西

- 葉脈（健康狀態下）
- 反光、水珠
- 灰塵、花粉
- 拍攝時的鏡頭髒點

### 四、品質門檻

| 指標 | 目標 |
|------|------|
| 跨標註者 IoU（5% 抽樣） | ≥ 0.55（病徵）；≥ 0.85（樹冠輪廓） |
| 漏標率 | < 5% |
| 誤標率 | < 5% |

## 伍、工作流

```
原始影像 → 標註者 A 標註 → 隨機 5% 進入 B 複標 → 計算 IoU
                                                 ↓
                                              < 門檻
                                                 ↓
                                           討論 → 修訂規範
```

### 工具設定

- **CVAT**：建立 project，按上述類別設置 label，使用 `polygon` 工具
- 匯出格式：COCO 1.0
- 匯入 TreeVision：`scripts/import_coco.py`
