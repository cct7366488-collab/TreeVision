# 野外影像採集標準作業 SOP（ML 可用性導向）

| 版本 | v0.1 |
|------|------|
| 適用 | TreeVision 土肉桂試驗固定攝點拍攝 |
| 目的 | 確保每張影像「可重現、可量測、可比較」，達到模型訓練可用標準 |
| 相關 | [pilot-protocol.md](pilot-protocol.md)（當日工序）、[ADR-0005](decisions/0005-fixed-camera-station-sop.md)（攝點規格）|

> **核心觀念**：模型只能從影像裡「真實存在且一致」的訊號學習。模糊、過曝、角度每次不同的影像，會把真正的生長訊號淹沒在拍攝雜訊裡，導致模型學不到、或學錯。**拍攝品質直接決定模型上限。**

---

## 壹、三大可用性前提（任何一張影像都要滿足）

一、**可重現**：同一株、每次回到同一個標樁、同攝距、同高度、同角度 → 影像差異 ≈ 樹的真實變化。

二、**可量測**：每張關鍵影像含尺度物（A4 方格紙／硬幣／鋼尺），否則葉面積、株高無法換算實際尺寸。

三、**可比較**：固定白平衡與曝光策略 → 葉色、病徵跨時序可比；自動模式每張跳動則不可比。

---

## 貳、相機設定（拍攝前一次設定，全程不變）

| 項目 | 設定 | 為什麼 |
|------|------|--------|
| 解析度 | 最高（長邊 ≥ 3000 px，至少 1500 px）| 葉脈/病斑需足夠像素才辨識得出 |
| 對焦 | 點對焦於主體（全株→主幹中段；葉片→葉面），半按鎖焦 | 自動對焦常對到背景 |
| 曝光 | 寧可輕微欠曝（可救），**絕不過曝**（葉面溢白＝病徵/葉色永久喪失）；必要時 EV −0.3 | 過曝資訊不可逆 |
| ISO | 固定低值（100–400）| 高 ISO 雜訊干擾紋理 |
| 白平衡 | 固定「日光」模式，**勿用自動** | 自動 WB 每張色溫跳動 → 葉色不可比 |
| 格式 | JPEG 最高品質（或 RAW+JPEG）| — |
| 關閉 | HDR、濾鏡、美顏、數位變焦 | HDR/濾鏡會竄改真實葉色與紋理 |
| 鏡頭 | 拍前擦淨 | 指紋/霧氣＝整批模糊 |

> 每場次先拍一張**色卡/灰卡**，作為事後色彩校正基準。

## 參、固定攝點架設（每株、每攝點）

一、三腳架腳位對準該方位**標樁**（北＝紅、東＝黃、葉片＝藍），誤差 < 5 cm。

二、**攝距**：幼齡林 2.5 m（雷射測距儀校正並記錄）；成熟林 5–7 m。

三、**相機高度**：鏡頭中心離地 1.5 m（捲尺量），三腳架水平儀調平。

四、**光軸**：全株攝點鏡頭水平、正對樹幹；葉片近拍鏡頭垂直紙面俯視。

五、回到同一樁、同高、同距、同角 —— 這是「可比較」的根本，不可省。

---

## 肆、立體感與多視角（為什麼一株要多個攝點）

一、單張正面平拍會把樹「壓平」，被遮擋的枝葉與深度資訊喪失，模型學不到株型與分枝結構。

二、**多攝點環繞**（核心北/東＋加值南/西/俯拍）還原樹體 3D 結構；同株各攝點視野保留重疊，利於日後 3D／光達式重建（加值）。

三、**讓影像有立體層次的技巧**：

（一）採**側光**（上午 9–11 點斜射光）讓枝葉產生陰影層次，看得出深度與紋理；避免正午頂光（扁平）與正面閃燈（打死陰影）。

（二）**背景單純**（天空或遠景），主體與背景分離，利於分割模型抓邊界。

（三）主體**填滿但不切邊**，搭配淺景深讓主體從背景跳出（但仍須全株清晰對焦）。

---

## 伍、各影像類型構圖規範

一、**whole_plant 全株**：整株完整入鏡（**含頂梢與基部**），主體占畫面 60–80%，四周留約 10% 邊。**不可切頂、切基**。成熟林框不下時 → 樹冠與主幹**分兩張**，各自完整。

二、**leaf_macro 葉片近拍**：

（一）取 5–10 片代表性葉（樹冠中段成熟葉 5 片＋含病徵 2–3 片）。

（二）平鋪 A4 方格紙，葉片**完整、不重疊、不折角**，葉柄朝同一方向。

（三）尺度物（方格紙邊框或硬幣）**完整入鏡**；鏡頭**垂直俯視**避免梯形變形；距 30–50 cm。

三、**scale_ref 尺度物**：每株或每場次至少一張清楚的尺度物特寫，供像素→公分換算校驗。

---

## 陸、光照與天氣

一、**最佳**：陰天散射光，或上午 9–11 點、下午 14–16 點斜射光（柔和、有方向性、有層次）。

二、**避免**：正午頂光（扁平無立體）、強逆光（主體剪影）、葉面直射反光、雨後水珠高光斑（擦乾或改期）。

三、同一樣區的一場次盡量在**相近時段**拍完（控光照一致）。

---

## 柒、現場即時品質自檢（每張放大檢查，不過就當場補拍）

一、對焦是否落在主體？放大看**葉脈是否清晰可辨**？

二、開**高光警示（斑馬紋）**：葉面是否過曝溢白？

三、整株是否**完整、無遮擋、無切邊**？

四、**尺度物**完整可見？

五、`station_id` / `tree_id` 記對？

> 離場後光照已變，無法補同一條件 → **當場補拍是鐵律**。

## 捌、不可用影像（NG）對照表

| NG 樣態 | 後果 | 正確做法 |
|---------|------|---------|
| 模糊（手震／未鎖焦）| 葉緣、病斑無法辨識 | 三腳架＋鎖焦＋連拍取最佳 |
| 過曝（頂光／閃燈）| 葉色與病徵永久喪失，不可救 | 寧可欠曝；側光／撐傘柔光 |
| 切頂／切基 | 株高、株型量測失準 | 退距或分張，確保完整 |
| 背景雜亂（他株／人／雜物）| 分割困難、誤判 | 換角度讓背景單純 |
| 每次角度/距離不同 | 時序不可比，生長訊號被淹沒 | 嚴格固定攝點 |
| 缺尺度物 | 葉面積/株高無法換算 | 每張關鍵影像入鏡尺度物 |
| 開 HDR／濾鏡／美顏 | 竄改真實葉色與紋理 | 全部關閉 |

## 玖、交付與入庫

一、每株每攝點**連拍 3 張**，現場標記最佳。

二、命名 `{station_id}_{YYYYMMDD}_{HHMM}_{seq}.jpg`（見 [SPEC §肆](SPEC.md)）。

三、當日上傳前跑**影像入庫管線**（`scripts/ingest_images.py`）預驗 metadata 與品質，產生待補拍清單，下次出場補齊。
