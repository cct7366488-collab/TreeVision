# web/ — 前端儀表板（骨架）

單檔、零建置的儀表板版型。**直接用瀏覽器開 `index.html` 即可看**。

## 設計

- **優先連 API**：載入時嘗試 `http://127.0.0.1:8000`（FastAPI 後端，見 [`../app/`](../app/)），
  成功則用即時資料、badge 顯示「● 連線 API」。
- **離線可看**：API 未連線時自動 fallback 到內嵌範例資料（取自本機 DB 真實查詢值），
  badge 顯示「範例資料」。版型在沒有後端時也完整可看。
- 可在頁尾改 API base 後按「重新連線」。

## 內容

| 區塊 | 說明 |
|------|------|
| KPI | 樣區 / 樣樹 / 處理組 / 量測筆數 |
| 試驗對照 | 各處理組平均胸徑 × 季別（2×2 修剪×施肥）|
| 處理組 × 樣區 | 樹數矩陣 |
| 場域概覽 | 4 樣區基本資料 |
| 樹木履歷 | 單株（多幹）歷次量測 |

## 本機預覽（連 API）

```powershell
# 1) 起後端（含 DB）
uvicorn app.main:app
# 2) 起靜態伺服器看前端
python -m http.server 8137 --directory web
#    開 http://127.0.0.1:8137
```
