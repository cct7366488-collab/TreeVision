### 二、研究目的

透過兩年期、區集式試驗設計，結合修枝矮化處理與不同施肥量處理，達成：

1. 評估土肉桂之**枝葉生物量**、**精油產率**與**精油成分**之變化
2. 同步觀測**生長量變化**與**雨後逕流環境影響**
3. 建立土肉桂人工林**生產管理基礎科學數據**
4. 作為**推廣栽培與產業升級**之依據

| 國內現有規模 | 花蓮 ≥ 100 ha、南投 ≥ 30 ha；南投年產值估 2 億元 |

## 參、試驗設計（CRBD）

### 一、總體框架

採用**完全隨機區集設計（Completely Randomized Block Design, CRBD）**，含兩個主要處理因子：

1. 修枝矮化處理（Pruning）
2. 施肥處理（Fertilization）

> 區集（block）= 樣區，藉以降低空間環境差異對處理效果的干擾。

### 二、處理因子定義

#### (一) 修枝矮化處理

| 屬性 | 內容 |
|------|------|
| 操作 | 剪除頂芽 + 適度疏除側枝 |
| 強度 | 控制株高在合理範圍，保留足夠葉量供生長 |
| 預期效應 | 解除頂端優勢 → 促進側枝萌發 → 增加葉片生物量 + 次級代謝物累積 |
| 對照組 | 不修剪（自然生長） |

#### (二) 施肥處理

| 屬性 | 內容 |
|------|------|
| 肥料組成 | **有機肥 : 矽肥 = 10 : 1（重量比）混合** |
| 矽肥商品 | 矽力康 |
| 施肥量等級 | 每株每次 150g（+ 矽肥 15g）/ 200g（+ 矽肥 20g）/ 250g（+ 矽肥 25g） |
| 對照組 | 不施肥（20 年生樣區設）/ 視幼齡林情況設定 |
| 施肥頻率 | 每年兩次（春季、雨季各一次） |
| 施肥方式 | **環狀溝施 / 穴施**，均勻施入樹冠垂直投影範圍土壤中 |
| 矽肥目的 | 增強土壤保肥性 + 植株抗性 + 促進根系發育 + 提高產量 |

### 三、處理排列（兩種林齡分開設計）

#### (一) 20 年生樣區（115-12-1）— 單因子試驗 + 對照

| 處理組合 | 株數 |
|---------|------|
| 修剪 + 施肥 | 30 |
| 修剪 + 不施肥 | 30 |
| 不修剪 + 施肥 | 30 |
| 不修剪 + 不施肥 | 30 |
| **合計** | **120 株** |

> 嚴格來說是 **2×2 = 4 組合**，每組 30 株重複個體。

#### (二) 幼齡林樣區（117-28-1, 1-2-3, 8-1-10）— 雙因子析因設計（Factorial）

每樣區為一個區集，內含 6 組處理組合：

| 修剪 | 施肥量 | 處理代號 |
|------|--------|---------|
| 是 | 150 g | P+F150 |
| 是 | 200 g | P+F200 |
| 是 | 250 g | P+F250 |
| 否 | 150 g | P-F150 |
| 否 | 200 g | P-F200 |
| 否 | 250 g | P-F250 |

每組約 10 株 → 每樣區 60 株 → **3 樣區共 180 株**

> 完全隨機區集設計確保不同地點間處理效果的可比性，並增加試驗結果的推廣性。

### 四、試驗時序

| 時點 | 動作 |
|------|------|
| 試驗開始前 | 所有樣樹編號標記、量測初始樹高與 DBH、記錄健康狀況 |
| 試驗第 1 季開始時 | **修枝處理一次性實施**（後續每年生長季末視需要進行輕度補剪） |
| 第 1 年春季 | 第 1 次施肥 |
| 第 1 年雨季 | 第 2 次施肥 |
| 第 2 年春季 | 第 3 次施肥 |
| 第 2 年雨季 | 第 4 次施肥 |
| 各季 | 生長量調查（每季一次） |
| 第 1、2 年生長季末 | 枝葉生物量採樣 + 精油萃取與分析 |

## 肆、調查項目與統計分析

### 一、調查項目

| # | 項目 | 頻率 | 量測內容 | 工具 |
|---|------|------|---------|------|
| 1 | 生長量 | 每季一次 | 樹高、胸高直徑（DBH，1.3 m 處）/ 地徑 | 測高器、直徑捲尺、測微尺 |
| 2 | 枝葉生物量 | 每年生長季結束時 | 鮮重 → 烘乾換算含水率 → 乾重 | 田秤、烘箱 |
| 3 | 精油含量 | 每年採收後 | 水蒸氣蒸餾法（Steam Distillation） | 蒸餾器、油水分離器 |
| 4 | 精油成分 | 每年採收後 | GC-MS 或 GC-FID | 屏東科技大學分析實驗室 |
| 5 | 土壤養分 | 試驗前 + 試驗結束時 | 有機質、全氮、有效磷、有效鉀、pH 值 | 標準土壤分析 |
| 6 | 雨後逕流（僅幼齡林） | 雨季大雨事件後 | 泥沙、養分濃度 | 表層逕流人工收集 |
| 7 | 植株健康狀況 | 試驗前基準 + 各季 | 主觀觀察 | 田間紀錄 |

> **本研究不列入記錄**：枝條長度、葉面積指數（為減少測量複雜度）。
> → **TreeVision 的機會**：影像 AI 可補上這兩項既未列入又有價值的指標。

### 二、統計分析方法

| 階段 | 方法 | 用途 |
|------|------|------|
| 1 | 單因子 / 雙因子 ANOVA | 檢驗處理組差異是否達統計顯著（α = 0.05） |
| 2 | 多重比較（LSD / Tukey） | 找出哪兩組之間差異顯著 |
| 3 | 線性 / 非線性迴歸 | 量化施肥量 ↔ 生長量、修剪 ↔ 葉量的劑量效應關係 |
| 4 | 多元迴歸（資料允許時） | 同時納入多因子的綜合影響 |
| 5 | 假設檢查 | 常態性、同質變異性、獨立性；必要時轉換或非參數檢定 |

統計軟體：SPSS / R

### 三、預期成果

1. 修枝矮化處理 → 預計提高側枝與葉片數、增加單株枝葉生物量與精油產量；但**過重修剪**反而抑制生長
2. 施肥處理 → 預計顯著提高生長速率、枝葉產量；矽肥 + 有機肥可能產生**累積效應**（強化土壤肥力與抗逆性）
3. 施肥量可能存在**最佳值**（過量會邊際遞減）
4. 修剪 × 施肥可能存在**交互作用**（施肥對修剪植株的促生效果可能更強）

## 陸、矮化修剪技術（PPTX 教材）

來源檔：`土肉桂矮化修剪技術.pptx`（16 張投影片）

### 一、矮化的目的（三大目的 + 一個底線）

| 目的 | 內容 |
|------|------|
| 1. 提高採收效率 | 矮化樹體 → 人工 / 機械採收更便捷安全 |
| 2. 增加葉片產量 | 刺激側枝萌發 → 顯著增加枝葉數量 → 提升單位面積產量 |
| 3. 降低災害風險 | 降低重心與受風面積 → 抵禦颱風天災 |
| **底線** | **避免毀滅性截幹**（成熟大樹進行毀滅性截幹會造成巨大難癒合傷口，導致主幹中空、弱化、甚至死亡） |

### 二、核心原理 — 頂端優勢（Apical Dominance）

| 階段 | 描述 |
|------|------|
| 修剪前 | 頂芽產生生長素 → 抑制側芽生長 → 養分主要向上 |
| 修剪後 | 移除頂芽 → 解除側芽抑制 → 側芽大量萌發 → 養分分散至各側枝 → 樹冠更茂密 → 葉片產量提升 |

> **TreeVision 影像辨識線索**：修剪後的植株會在原頂芽位置附近出現多個新生側枝（離切口 15~20 cm 內），這是模型可學習的典型特徵。

### 三、三大修剪技術

#### (一) 截幹（Heading / Truncation）— 矮化塑型第一步

| 項目 | 規範 |
|------|------|
| 適用時機 | 幼樹期，主幹直徑 ≤ 5 cm，傷口小、癒合快 |
| 截幹高度 | 人工採收：**離地 1~1.5 公尺**；機械採收：依機具操作高度 |
| 禁忌 | 切勿對成林大樹進行毀滅性截幹（會導致腐朽與死亡） |

#### (二) 打頂（Topping / Pinching）— 維持矮化樹型

| 項目 | 規範 |
|------|------|
| 操作 | 剪除頂芽或新梢 / 摘心，持續移除頂端優勢 |
| 頻率 | 採葉栽培本身即有持續打頂效果；徒長新枝應持續剪除 |
| 注意 | 確保充足肥水管理；管理不當會導致樹勢衰弱 |

#### (三) 三切法（Three-cut Method）— 粗枝下刀，傷口最小化

| 步驟 | 操作 |
|------|------|
| 第一切（預防） | 距主幹 20~30 cm，由下往上切入 1/3 |
| 第二切（卸重） | 在第一切外側 2~5 cm，由上往下切斷 |
| 第三切（收尾） | 沿**枝領（Branch Collar）**外緣切除殘樁，確保切口平整 |

> **枝領** = 枝條基部隆起，含抗病物質，是癒合關鍵部位，**絕不可傷害**。

### 四、田間修剪 SOP

| 項目 | 規範 |
|------|------|
| 工具選擇 | 細枝用修枝剪、粗枝用手鋸、高處用高枝剪 |
| 工具消毒 | 修剪前後以**酒精或稀釋漂白水消毒**，防病菌傳播 |
| 切口角度 | 在芽點外側約 **1 cm** 處，**45° 角斜切**（助雨水滑落，防積水腐爛） |
| 修剪順序 | **「由基至梢，由內到外」**：先處理基部與內側，再修剪末梢與外圍 |

### 五、最佳修剪時機

| 季節 | 適合性 | 理由 |
|------|--------|------|
| **春節後 ~ 清明前**（黃金期） | ✓ 最佳 | 土肉桂為休眠至萌芽初期；樹液流動緩慢，不易流失養分；癒合快 |
| 5~7 月（生長旺季） | ✗ 避免 | 樹液旺盛，修剪易傷樹勢 |
| 雨季 | ✗ 避免 | 潮濕環境增加傷口感染風險 |

### 六、採收型修剪規劃

| 項目 | 規範 |
|------|------|
| 一年次數 | **3~5 次 / 年**（採收即修剪） |
| 葉片品質最佳期 | **中秋節過後 ~ 隔年 3 月** |
| 注意事項 | 避開雨天；依樹勢觀察調整修剪頻率 |

### 七、修剪後的肥培

二年生以上植株每年**追施三次肥料**：

| 月份 | 動作 |
|------|------|
| 2 月 | 第一次施肥 |
| 6 月 | 第二次施肥 |
| 10 月 | 第三次施肥 |

| 肥料 | 規範 |
|------|------|
| 種類 | 氮磷鉀均衡複合肥（如 20-5-10），或適量尿素 / 人糞尿 |
| 施肥方式 | **環形溝施肥**（樹冠投影外緣開溝） |
| 注意 | 施用乾肥後若天乾需澆水 |

> ⚠️ 注意：報告書（DOCX）規範的施肥**為兩年期試驗特定設計**（每年 2 次，有機肥 + 矽肥 10:1，150/200/250g），
> 而 PPTX 教材中的「每年 3 次（2/6/10 月）」是一般林農推廣建議。
> **TreeVision 應以 DOCX 試驗設計為主，PPTX 教材作為實務知識背景**。

### 八、現場健康評估 — 5 個快速指標

對應到 TreeVision 影像分析的健康度量化（從主觀 → 客觀化）：

| # | 現場觀察指標 | TreeVision 影像對應指標 |
|---|------------|----------------------|
| 1 | 葉片狀態（顏色、質地、有無黃化、枯萎、病斑） | green_index_mean, yellowing_ratio, disease_lesion_ratio |
| 2 | 新芽與分枝（萌發情形、是否均勻旺盛） | new_leaf_count, new_leaf_ratio, branch_count（v1+） |
| 3 | 傷口癒合（樹幹、枝條修剪傷口） | wound_segmentation（特化模型，v2+） |
| 4 | 根部狀況（積水、腐爛） | （影像難以涵蓋，仍需現場記錄） |
| 5 | 整體樹勢（生長速度、葉片產量、抗逆性） | 綜合 health_score |

### 九、踩雷案例（避免錯誤）

| 錯誤行為 | 後果 |
|---------|------|
| 毀滅性截幹 | 巨大傷口 → 腐朽 → 結構弱化 |
| 時機不當（生長旺季 / 雨季修剪） | 樹液流失、感染 |
| 工具未消毒 / 切口不平整 | 病菌入侵風險 |
| 修剪後未加強肥培 | 樹勢衰弱 |

## 柒、土肉桂物種特性（DOCX 附錄一摘要）

### 一、生態特性

| 屬性 | 內容 |
|------|------|
| 學名 | *Cinnamomum osmophloeum* Kanehira |
| 中文名 | 臺灣土肉桂 |
| 分類 | 樟科常綠中喬木（臺灣特有種） |
| 分布海拔 | 中低海拔山區，**500~2,000 公尺**之闊葉林區 |
| 樹皮 | 平滑、富粘質、有濃烈肉桂香味 |
| 葉 | **互生或近對生，革質**，卵形 / 卵狀長橢圓形，**長 8~12 cm**，**三出脈明顯**，下表面粉白 |
| 花 | 聚繖花序腋生及頂生，長約 10 cm，白色絹毛 |
| 果 | 核果橢圓形，長約 1 cm |
| 與胡氏肉桂區別 | 土肉桂頂芽**無芽鱗**且葉、樹皮均有濃郁肉桂味 |
| 與陰香區別 | 土肉桂幼枝**綠色**、葉背**粉白**；陰香幼枝紅色、葉背青綠 |

### 二、化學品系（6 型）

由葉子精油組成可分為：

| 化學品系 | 特徵氣味 | 主要成分 |
|---------|---------|---------|
| 肉桂醛型（Cinnamaldehyde type） | 濃郁肉桂香味 | 肉桂醛佔 ≥ 50% |
| 伽羅木醇型（Linalool type） | 強烈花香 + 水果香（甜橘香） | S-(+)-Linalool |
| 樟腦型（Camphor type） | 樟腦味 | Camphor |
| 肉桂醛-肉桂乙酸酯型 | — | Cinnamaldehyde + Cinnamyl acetate |
| 肉桂乙酸酯型 | — | Cinnamyl acetate |
| 混合型 | — | 多成分混合 |

> **TreeVision 應預留化學品系欄位**（雖然 v1 不做樹種辨識，但同樹種不同化學型可能有外觀差異）。

### 三、栽培環境需求

| 項目 | 範圍 |
|------|------|
| 最佳溫度 | 20~28°C |
| 年降雨量 | ≥ 1,500 mm |
| 土壤 pH | 5.5~6.5（最佳） |
| 排水 | 良好（避免積水導致根腐病） |
| 行株距 | 行距 3~4 m、株距 2~3 m |
| 一般密度 | 約 1,500 株 / ha |

### 四、葉片採收最佳期

| 階段 | 時機 |
|------|------|
| 開始採收 | 樹齡 **3~5 年** |
| 最佳季節 | **旱季結束、雨季開始之前** |
| 葉片選擇 | 成熟、色澤鮮亮 |
| 採摘方向 | **從下往上**（保護植株） |
| 天氣 | 晴朗、乾燥日 |

### 五、葉片乾燥處理

| 方法 | 規範 |
|------|------|
| 自然風乾 | 均勻鋪開、乾燥通風處、避免陽光直射 |
| 人工乾燥 | 乾燥機，**40~50°C**，適當通風 |
| 後處理 | 篩選破損 → 分類打包 |

| 指標 | 量測方式 | TreeVision 是否能輔助 |

### 三、SOP 對齊（必須遵循）

1. **場域編號對齊**：site_id 採官方林班作業列區編號（如 `115-12-1`、`117-28-1`、`1-2-3`、`8-1-10`），不自編
2. **時序密度對齊**：與官方「每季調查」對齊，影像拍攝排程**至少季配**（春末、夏末、秋末、冬末），密集期可月配
3. **處理組編碼對齊**：treatment_id 採 P+/P- × F0/F150/F200/F250 編碼
4. **健康等級對齊**：仍保留官方 1~5 主觀評分欄位（作為人工標註的 ground truth），TreeVision 額外輸出客觀指標
5. **危害分類對齊**：採官方 9 類干擾型態 enum 列舉
